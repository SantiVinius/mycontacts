# API com NodeJS
API para criar, listar, editar e deletar um registro, usando Node + Express.

## Tecnologias

* [NodeJS](https://nodejs.org/en/)
* [Docker](https://www.docker.com/)
* [PostgresSQL](https://hub.docker.com/_/postgres)
* [Express](https://expressjs.com/pt-br/)
